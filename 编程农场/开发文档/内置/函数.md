
## 1. API 全函数参考

### 物理操作 (200 Ticks 消耗)

- **`move(dir)`**：向指定方向移动，出界会自动绕回（Wrap）。
- **`harvest()`**：收割实体。若不可收割则会直接摧毁该地块。
- **`plant(ent)`**：消耗资源种植，地块错误或已有植物会失败。
- **`till()`**：将 `Grassland` 与 `Soil` 状态互换。
- **`swap(dir)`**：与相邻格交换位置，排序算法的核心函数。
- **`use_item(item, n)`**：使用肥料、水或神秘物质。

### 环境感知 (1 Tick 消耗)

- **`can_harvest()`**：检测成熟状态。南瓜枯萎时返回 `False`。
- **`get_entity_type()`**：获取地块实体，空地块返回 `None`。
- **`get_ground_type()`**：检测是草地还是土壤。
- **`get_pos_x() / get_pos_y()`**：获取当前坐标坐标（下标从 0 开始）。
- **`get_world_size()`**：获取地图边长 $N$。
- **`measure(dir)`**：测量向日葵花瓣、仙人掌尺寸或迷宫宝藏位置。
