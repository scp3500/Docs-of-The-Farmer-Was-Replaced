# 元组
元组是将多个值合并为单一值的好方法。
要创建一个元组，只需用逗号分隔值：

`tuple = 1, 2`

你也可以将它们再次解包到多个变量中。在下面的代码中，元组 `(1,2)` 被解包到 2 个变量 `a` 和 `b` 中。

`a, b = 1, 2`

元组可以像列表一样进行索引，但它们是不可变的，创建后不能更改。

`tuple = 1, 2`

`print(tuple[1])`
打印 `2`

`tuple[0] = 3`
抛出错误

与列表不同的是，元组可以被作为字典中的`key`进行使用。

`d = {(1,2):(4,5)}

print(d[(1,2)])`
打印 `(4,5)`

元组也可用于在函数中返回多个值。

`def f():
    return 1, 2

a, b = f()`
